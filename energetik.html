🌍 Systeme und Umgebung

Ein System ist ein festgelegter räumlicher Bereich, in dem sich Stoffe befinden oder eine chemische Reaktion abläuft. Alles andere ist die Umgebung.

🔒 Isoliertes System

Kann weder Stoffe noch Energie mit der Umgebung austauschen.

Beispiel: Thermosflasche (idealisiert)

🚪 Geschlossenes System

Kann nur Energie (nicht Stoffe) mit der Umgebung austauschen.

Beispiel: Verschlossener Kolben mit beweglichem Stempel

🌬️ Offenes System

Kann Stoffe und Energie mit der Umgebung austauschen.

Beispiel: Becher mit chemischer Reaktion

💡 Wichtige Erkenntnis:

Die meisten chemischen Reaktionen laufen in offenen Systemen bei konstantem Druck ab. Deshalb ist die Enthalpie oft wichtiger als die innere Energie!

🔋 Innere Energie und Enthalpie

🔋 Innere Energie (U)

Die innere Energie U ist die Summe aller Energien der Teilchen in einem System:

  • Kinetische Energie der Teilchen
  • Potentielle Energie der Teilchen
  • Bindungsenergie in Molekülen
  • Elektronische Energie in Atomen
Reaktionsenergie
ΔᵣU = U₂ - U₁ = Qᵥ
Bei konstantem Volumen

⚡ Enthalpie (H)

Die Enthalpie H beschreibt Energieänderungen bei konstantem Druck:

Definition der Enthalpie
H = U + p·V
Reaktionsenthalpie
ΔᵣH = H₂ - H₁ = Qₚ
Bei konstantem Druck
Zusammenhang U und H
ΔᵣH = ΔᵣU + p·ΔV

🔥 Exotherm vs. Endotherm

Exotherm: ΔᵣH < 0
Energie wird freigesetzt
Beispiel: Verbrennung
Endotherm: ΔᵣH > 0
Energie wird verbraucht
Beispiel: Photosynthese
🌡️ Kalorimetrie

Die Kalorimetrie ist die experimentelle Bestimmung von Reaktionswärmen mithilfe eines Kalorimeters.

Grundgleichung der Kalorimetrie
Qᵣ = -(cw · mw + Cc) · Δϑ
Symbol Bedeutung Einheit
Qᵣ Reaktionswärme J
cw Spezifische Wärmekapazität des Wassers 4,18 J/(g·K)
mw Masse des Wassers g
Cc Wärmekapazität des Kalorimeters J/K
Δϑ Temperaturänderung K
🧪 Experimentelle Hinweise:
  • Kalorimeter muss gut isoliert sein
  • Vollständige Durchmischung ist wichtig
  • Temperatur kontinuierlich messen
  • Wärmekapazität des Kalorimeters berücksichtigen
🏗️ Bildungs- und Verbrennungsenthalpien

🏗️ Bildungsenthalpie

Die molare Standardbildungsenthalpie ΔfH°m ist die Enthalpieänderung bei der Bildung von 1 mol einer Verbindung aus den Elementen bei Standardbedingungen.

Standardbedingungen
ϑ = 25°C (298 K)
p = 1000 hPa = 1 bar
📌 Wichtig: ΔfH°m(Elemente) = 0 kJ/mol

🔥 Verbrennungsenthalpie

Die molare Standardverbrennungsenthalpie ΔcH°m ist die Enthalpieänderung bei der vollständigen Verbrennung von 1 mol einer Verbindung.

Beispiel: Methan

CH₄ + 2O₂ → CO₂ + 2H₂O

ΔcH°m(CH₄) = -890 kJ/mol

📊 Wichtige Standardbildungsenthalpien (kJ/mol)

Verbindung ΔfH°m Verbindung ΔfH°m
H₂O(l) -286 CO₂(g) -394
H₂O(g) -242 NH₃(g) -46
HCl(g) -92 CH₄(g) -75
NaCl(s) -411 C₂H₅OH(l) -278
🔄 Satz von Hess

📜 Satz von Hess

Die Reaktionsenthalpie ist unabhängig vom Reaktionsweg.

Berechnung mit Bildungsenthalpien
ΔᵣH° = Σ νᵢ·ΔfH°m(Produkte) - Σ νᵢ·ΔfH°m(Edukte)

🧩 Übungsaufgabe: Verbrennung von Ethanol

Aufgabe: Berechne die Standardreaktionsenthalpie für die Verbrennung von Ethanol:

C₂H₅OH(l) + 3O₂(g) → 2CO₂(g) + 3H₂O(l)

Gegeben:

  • ΔfH°m(C₂H₅OH) = -278 kJ/mol
  • ΔfH°m(CO₂) = -394 kJ/mol
  • ΔfH°m(H₂O) = -286 kJ/mol
  • ΔfH°m(O₂) = 0 kJ/mol

Lösung:

ΔᵣH° = [2·(-394) + 3·(-286)] - [1·(-278) + 3·0]

ΔᵣH° = [-788 - 858] - [-278]

ΔᵣH° = -1646 + 278 = -1368 kJ/mol

🌪️ Entropie

Die Entropie S ist ein Maß für die Unordnung oder Zufallsverteilung der Teilchen in einem System.

📈 Entropie nimmt zu bei:

  • Temperaturerhöhung
  • Zunahme der Teilchenzahl
  • Phasenübergängen (s→l→g)
  • Vermischung von Stoffen

📉 Entropie nimmt ab bei:

  • Abkühlung
  • Kondensation/Erstarrung
  • Bildung geordneter Strukturen
  • Abnahme der Teilchenzahl
Standardreaktionsentropie
ΔᵣS° = Σ νᵢ·S°m(Produkte) - Σ νᵢ·S°m(Edukte)

🌟 Zweiter Hauptsatz der Thermodynamik

Die Gesamtentropie (System + Umgebung) kann nur zunehmen oder gleich bleiben:

ΔSGesamt ≥ 0

Spontane Vorgänge sind immer mit einer Zunahme der Gesamtentropie verbunden!

H₂O(l) 70 H₂O(g) 189

🆓 Freie Enthalpie (Gibbs-Energie)

Die freie Enthalpie G (Gibbs-Energie) kombiniert Enthalpie und Entropie und bestimmt, ob eine Reaktion spontan abläuft.

Gibbs-Helmholtz-Gleichung
ΔᵣG = ΔᵣH - T·ΔᵣS
Zusammenhang mit Gesamtentropie
ΔᵣG = -T·ΔSGesamt

🎯 Kriterien für Spontaneität

✅ Exergonisch (spontan)

ΔᵣG < 0

Reaktion läuft freiwillig ab

Gleichgewicht liegt bei den Produkten

❌ Endergonisch (nicht spontan)

ΔᵣG > 0

Reaktion läuft nicht freiwillig ab

Gleichgewicht liegt bei den Edukten

⚖️ Gleichgewicht

ΔᵣG = 0

System im Gleichgewicht

Edukte ⇌ Produkte

🌡️ Temperaturabhängigkeit der Spontaneität

ΔᵣH ΔᵣS ΔᵣG bei niedriger T ΔᵣG bei hoher T Spontaneität
< 0 > 0 < 0 < 0 Immer spontan
> 0 < 0 > 0 > 0 Nie spontan
< 0 < 0 < 0 > 0 Nur bei niedriger T
> 0 > 0 > 0 < 0 Nur bei hoher T

🧩 Übungsaufgabe: CaCO₃-Zersetzung

Aufgabe: Bei welcher Temperatur zersetzt sich CaCO₃ spontan?

CaCO₃(s) → CaO(s) + CO₂(g)

Gegeben:

  • ΔᵣH° = +178 kJ/mol
  • ΔᵣS° = +161 J/(mol·K)

Lösung:

Bei ΔᵣG = 0 gilt: T = ΔᵣH° / ΔᵣS°

T = 178.000 J/mol ÷ 161 J/(mol·K)

T = 1106 K = 833°C

Oberhalb von 833°C zersetzt sich CaCO₃ spontan!

🔬 Anwendungen und Beispiele

🏭 Technische Anwendungen

🔋 Galvanische Zellen

Die maximale Arbeit einer Batterie entspricht ΔᵣG der ablaufenden Reaktion:

Wmax = -ΔᵣG

🏗️ Metallgewinnung

Reduktion von Metalloxiden ist nur bei genügend hoher Temperatur spontan (ΔᵣG < 0).

🧪 Phasenübergänge

Schmelz- und Siedetemperaturen entsprechen ΔᵣG = 0 für den Phasenübergang.

🌿 Biologische Systeme

ATP-Hydrolyse: ATP + H₂O → ADP + Pᵢ

ΔᵣG° = -31 kJ/mol

Diese exergonische Reaktion treibt viele endergonische Prozesse in Zellen an!

⚖️ Gleichgewichtslage vorhersagen

Zusammenhang ΔᵣG° und Gleichgewichtskonstante
ΔᵣG° = -RT·ln(K)
R = 8,314 J/(mol·K)

📝 Zusammenfassung und Übungen

🎯 Die wichtigsten Punkte auf einen Blick:

📚 Grundbegriffe:

  • System vs. Umgebung
  • Innere Energie U
  • Enthalpie H = U + pV
  • Entropie S (Unordnung)
  • Freie Enthalpie G

🔑 Schlüsselgleichungen:

  • ΔᵣH° = Σ ΔfH°(P) - Σ ΔfH°(E)
  • ΔᵣG = ΔᵣH - T·ΔᵣS
  • Spontan wenn ΔᵣG < 0
  • Qᵣ = -(cw·mw + Cc)·Δϑ

🧪 Weitere Übungsaufgaben

1. Berechne ΔᵣH° für: N₂ + 3H₂ → 2NH₃ (ΔfH°(NH₃) = -46 kJ/mol)

ΔᵣH° = 2·(-46) - [0 + 0] = -92 kJ/mol

2. Ist die Verdampfung von Wasser bei 25°C spontan? (ΔᵥₐₚH = 44 kJ/mol, ΔᵥₐₚS = 119 J/(mol·K))

ΔG = 44.000 - 298·119 = 44.000 - 35.462 = +8.538 J/mol

Da ΔG > 0: Nicht spontan bei 25°C

3. Ein Kalorimeter enthält 200 g Wasser. Die Temperatur steigt um 5,2 K. Berechne die freigesetzte Wärme (Cc = 50 J/K).

Qᵣ = -(4,18·200 + 50)·5,2

Qᵣ = -(836 + 50)·5,2 = -4.607 J

4,6 kJ wurden freigesetzt (exotherm)

🎯 Abitur-Tipps:

  • Einheiten beachten: kJ/mol vs. J/(mol·K)
  • Vorzeichen: Exotherm = negativ, Endotherm = positiv
  • Standardbedingungen: 25°C, 1 bar
  • Formelsammlung: Alle Formeln stehen drin!
  • Rechenwege: Immer klar strukturieren
