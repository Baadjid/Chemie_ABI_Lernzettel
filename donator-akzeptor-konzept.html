🔄 Donator-Akzeptor-Konzept

Ein universelles Prinzip für Säure-Base- und Redoxreaktionen

Basiskonzept 3 von 5 | Abitur 2026

🎯 Grundprinzip des Donator-Akzeptor-Konzepts

📚 Definition

Das Donator-Akzeptor-Konzept beschreibt chemische Reaktionen als Übertragung von Teilchen zwischen einem Donator (Geber) und einem Akzeptor (Empfänger). Dieses universelle Prinzip erklärt sowohl Säure-Base-Reaktionen (Protonenübertragung) als auch Redoxreaktionen (Elektronenübertragung).

💡 Merksatz

Donator = Geber | Akzeptor = Empfänger

Jede Donator-Akzeptor-Reaktion führt zur Bildung korrespondierender Paare!

🔄 Protonenübertragung

Säure-Base-Reaktionen

Übertragung von H⁺-Ionen (Protonen) vom Säure-Donator zum Base-Akzeptor

⚡ Elektronenübertragung

Redoxreaktionen

Übertragung von Elektronen vom Reduktionsmittel-Donator zum Oxidationsmittel-Akzeptor

⚖️ Säure-Base- vs. Redoxreaktionen im Vergleich
Kriterium 🧪 Säure-Base-System ⚡ Redoxsystem
Übertragene Teilchen Protonen (H⁺) Elektronen (e⁻)
Donator Säure (HA) Reduktionsmittel (Red)
Akzeptor Base (A⁻) Oxidationsmittel (Ox)
Korrespondierende Paare HA/A⁻ und HB⁺/B Red₁/Ox₁ und Red₂/Ox₂
Stärkemaß pKₛ-Wert Standardpotential E°
Gleichung Henderson-Hasselbalch Nernst-Gleichung

⚠️ Wichtig für das Abitur

  • Beide Reaktionstypen folgen dem gleichen Donator-Akzeptor-Prinzip
  • Korrespondierende Paare sind immer gekoppelt
  • Starke Donatoren haben schwache korrespondierende Akzeptoren
🧪 Säure-Base-Reaktionen nach Brønsted

Brønsted-Definition (1923)

🔴 Säuren (Protonendonatoren)

Säuren sind Teilchen, die Protonen (H⁺) abgeben können.

HA → H⁺ + A⁻
(Säure → Proton + konjugierte Base)

🔵 Basen (Protonenakzeptoren)

Basen sind Teilchen, die Protonen (H⁺) aufnehmen können.

B + H⁺ → BH⁺
(Base + Proton → konjugierte Säure)

🔄 Korrespondierende Säure-Base-Paare

Jede Säure hat eine korrespondierende Base und umgekehrt:

HA/A⁻ (Säure-Base-Paar 1)
HB⁺/B (Säure-Base-Paar 2)

Typische Säure-Base-Reaktionen

1. Salzsäure + Wasser
HCl + H₂O ⇌ H₃O⁺ + Cl⁻
Säure₁ + Base₂ ⇌ Säure₂ + Base₁
2. Ammoniak + Wasser
NH₃ + H₂O ⇌ NH₄⁺ + OH⁻
Base₁ + Säure₂ ⇌ Säure₁ + Base₂
3. Essigsäure + Wasser
CH₃COOH + H₂O ⇌ CH₃COO⁻ + H₃O⁺
Säure₁ + Base₂ ⇌ Base₁ + Säure₂

🌊 Wasser als Ampholyt

Wasser kann sowohl als Säure (Protonendonator) als auch als Base (Protonenakzeptor) wirken!

Säure- und Basenstärke

pKₛ-Wert als Stärkemaß

Je kleiner der pKₛ-Wert, desto stärker die Säure.

Kₛ = [H₃O⁺] × [A⁻] / [HA]
pKₛ = -log(Kₛ)
Säure pKₛ-Wert Stärke Konjugierte Base
HCl -6 sehr stark Cl⁻ (sehr schwach)
H₂SO₄ -3 sehr stark HSO₄⁻ (sehr schwach)
CH₃COOH 4,75 schwach CH₃COO⁻ (mittel)
NH₄⁺ 9,25 sehr schwach NH₃ (stark)
⚡ Redoxreaktionen als Elektronenübertragung

Elektronenübertragung bei Redoxreaktionen

🔴 Oxidation (Elektronenabgabe)

Das Reduktionsmittel gibt Elektronen ab und wird dabei selbst oxidiert.

Red → Ox + e⁻
Elektronendonator

🔵 Reduktion (Elektronenaufnahme)

Das Oxidationsmittel nimmt Elektronen auf und wird dabei selbst reduziert.

Ox + e⁻ → Red
Elektronenakzeptor

🔄 Korrespondierende Redoxpaare

Jedes Reduktionsmittel hat ein korrespondierendes Oxidationsmittel:

Red₁/Ox₁ (Redoxpaar 1)
Red₂/Ox₂ (Redoxpaar 2)
Beispiel: Zink + Kupfersulfat
Zn + Cu²⁺ → Zn²⁺ + Cu

Teilreaktionen:
Oxidation: Zn → Zn²⁺ + 2e⁻
Reduktion: Cu²⁺ + 2e⁻ → Cu

Oxidationszahlen bestimmen

📋 Regeln für Oxidationszahlen

  1. Elementare Stoffe haben die Oxidationszahl 0
  2. Die Summe aller Oxidationszahlen = Ladung des Teilchens
  3. H hat meist +I, O meist -II
  4. Alkalimetalle haben +I, Erdalkalimetalle +II
  5. Halogene haben meist -I (außer in Sauerstoffverbindungen)
Oxidationszahlen-Beispiele:
H₂O: H(+I), O(-II)
CO₂: C(+IV), O(-II)
NH₃: N(-III), H(+I)
SO₄²⁻: S(+VI), O(-II)
MnO₄⁻: Mn(+VII), O(-II)

🔄 Oxidation und Reduktion erkennen

Oxidation: Oxidationszahl steigt (Elektronenverlust)
Reduktion: Oxidationszahl sinkt (Elektronengewinn)

Beispiel: Ethanol-Oxidation
CH₃CH₂OH + CuO → CH₃CHO + Cu + H₂O

Oxidationszahlen:
C in CH₃CH₂OH: -I → C in CH₃CHO: +I (Oxidation)
Cu in CuO: +II → Cu: 0 (Reduktion)

Elektrochemische Spannungsreihe

⚡ Standardpotentiale E°

Je negativer das Standardpotential E°, desto stärker das Reduktionsmittel.

Redoxpaar E° (V) Reduktionsmittel-Stärke
Li⁺/Li -3,04 sehr stark
K⁺/K -2,92 sehr stark
Zn²⁺/Zn -0,76 mittel
2H⁺/H₂ 0,00 Referenz
Cu²⁺/Cu +0,34 schwach
Ag⁺/Ag +0,80 sehr schwach
F₂/2F⁻ +2,87 sehr schwach

🎯 Reaktionsvorhersage

Eine Redoxreaktion läuft freiwillig ab, wenn ΔE° > 0 ist:

ΔE° = E°(Ox/Red)ₖₐₜₕₒᵈₑ - E°(Ox/Red)ₐₙₒᵈₑ

🧩 Interaktives Quiz: Donator-Akzeptor-Konzept

Frage 1: Säure-Base-Reaktionen

Was ist NH₃ in der Reaktion: NH₃ + H₂O ⇌ NH₄⁺ + OH⁻?

a) Säure (Protonendonator)
b) Base (Protonenakzeptor)
c) Weder Säure noch Base
d) Sowohl Säure als auch Base

Richtig! NH₃ nimmt ein Proton (H⁺) von Wasser auf und wird zu NH₄⁺. Damit ist NH₃ eine Base (Protonenakzeptor) nach Brønsted.

Frage 2: Redoxreaktionen

In der Reaktion Zn + Cu²⁺ → Zn²⁺ + Cu wird Zink:

a) Oxidiert (gibt Elektronen ab)
b) Reduziert (nimmt Elektronen auf)
c) Weder oxidiert noch reduziert
d) Protolysiert

Richtig! Zn wird zu Zn²⁺ und verliert dabei 2 Elektronen. Dies ist eine Oxidation. Zn ist das Reduktionsmittel (Elektronendonator).

Frage 3: Korrespondierende Paare

Welches ist das korrespondierende Säure-Base-Paar zu CH₃COOH/CH₃COO⁻?

a) H₂O/OH⁻
b) H₃O⁺/H₂O
c) NH₄⁺/NH₃
d) HCl/Cl⁻

Richtig! In der Reaktion CH₃COOH + H₂O ⇌ CH₃COO⁻ + H₃O⁺ sind die korrespondierenden Paare: CH₃COOH/CH₃COO⁻ und H₃O⁺/H₂O.
